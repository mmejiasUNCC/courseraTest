# Simple Interest Calculator
This project describes a simple interest calculator.
## Formula
Simple Interest (SI) = (P * R * T) / 100
Where:
P = Principal amount, R = Annual interest rate (percent), T = Time (years)

## Inputs:
Principal (P), Rate (R), Time (T)

## Output
Simple Interest (SI)

## Example
If P = 100, R = 5, T = 2:

SI = (100 * 5 * 2) / 100 = 10
